# Simple Roulette App

A simple multiplatform app for take random decisions, with the features to save multiple lists and personalize the experience and style
This app will work on Android, iOS and Web.

## Features
- Save list for future decisions
- Modes:
	- Delete winner item
	- Keep winner item
- Click or drag to spin
- Full opensource and offline
- Easy list splitting items with commas or line escapes

## Get started

1. Install dependencies

   ```bash
   pnpm install
   ```

2. Start the app

   ```bash
   pnpx expo start
   ```
